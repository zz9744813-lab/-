# Compass × Hermes

Phase 1 foundation scaffold for Compass personal growth OS.

## Run locally

```bash
pnpm install
pnpm db:migrate
pnpm dev
```

Default dev URL: `http://localhost:3001`

## Notes

- Compass is UI + local data body.
- Hermes is the AI brain.
- All AI flows must route through Hermes APIs/MCP.
